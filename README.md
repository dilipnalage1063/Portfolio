### 1. SAPT (Student Assessment and Performance Tracker)

**Problem Statement:**
Educational institutions often lack efficient, real-time tracking of student performance and automated communication with parents. SAPT was built to bridge this gap by providing a centralized dashboard for grade tracking and automated notifications.

**Architecture Diagram:**
![SAPT Architecture](https://placehold.co/600x400?text=SAPT+Architecture+Diagram)

**Features:**
- **Grade Management:** Centralized system for tracking student assessments.
- **Automated Notifications:** Integrated Twilio and Resend APIs for real-time SMS and Email alerts.
- **Performance Analytics:** Visual representation of student progress over time.
- **Efficiency:** Reduced bulk notification time by 40%.

**Tech Stack:**
- **Frontend:** React.js
- **Backend:** Java Spring Boot
- **Database:** MySQL
- **APIs:** Twilio (SMS), Resend (Email)

**Installation Steps:**
1. Clone the SAPT repository.
2. Configure MySQL database and update `application.properties`.
3. Run the Spring Boot backend using Maven.
4. Install frontend dependencies with `npm install` and run `npm run dev`.

### 2. AI SkillBridge

**Problem Statement:**
Students struggle to identify skill gaps and create personalized study plans that align with industry requirements. AI SkillBridge uses generative AI to analyze resumes and provide actionable study roadmaps.

**Architecture Diagram:**
![AI SkillBridge Architecture](https://placehold.co/600x400?text=AI+SkillBridge+Architecture)

**Features:**
- **AI Resume Analysis:** PDF resume parsing using Apache Tika and Gemini AI.
- **Personalized Study Plans:** Dynamic roadmap generation based on skill gaps.
- **High Availability:** Implemented fallback logic for AI services.
- **Scalability:** Containerized with Docker and deployed on Railway.

**Tech Stack:**
- **Frontend:** HTML, CSS, JavaScript (React)
- **Backend:** Java Spring Boot
- **AI:** Google Gemini AI
- **Tools:** Apache Tika, Docker, Railway

**Installation Steps:**
1. Clone the AI SkillBridge repository.
2. Set up Gemini AI API keys in environment variables.
3. Build the Docker image: `docker build -t ai-skillbridge .`.
4. Run the container or deploy to Railway.
